# 前端仪表盘需求文档

## 1. 后端API接口

### 1.1 仪表盘统计数据

### 1.2 销售趋势数据

**接口地址**: `GET /api/admin/statistics/sales-trend`

**请求参数**:
- `timeRange`: 时间范围，可选值：`7d`（近7天）、`30d`（近30天）、`90d`（近90天），默认值：`7d`

**返回数据结构**:
```json
{
  "code": 200,
  "data": {
    "dates": ["2026-04-19", "2026-04-20", ...], // 日期列表
    "sales": [0.0, 0.0, ...],                   // 每日销售额
    "orders": [0, 0, ...]                       // 每日订单数
  },
  "message": "success"
}
```

## 2. 前端实现建议

### 2.1 核心统计卡片

- 使用 `el-card` 组件实现统计卡片
- 从 `/api/admin/dashboard/statistics` 获取数据
- 绑定 `stats` 对象中的各个属性到对应卡片

### 2.2 销售趋势图表

- 使用 ECharts 实现折线图
- 从 `/api/admin/statistics/sales-trend` 获取数据
- 支持时间范围切换（7d、30d、90d）
- X轴显示日期，Y轴显示销售额

### 2.3 订单状态分布图表

- 使用 ECharts 实现饼图
- 从 `/api/admin/statistics/order-status` 获取数据
- 显示各状态订单的占比
- 鼠标悬停显示具体数值和占比

### 2.4 最近订单列表

- 使用 `el-table` 组件实现订单列表
- 从 `/api/admin/dashboard/statistics` 获取 `recentOrders` 数据
- 订单状态使用 `el-tag` 组件显示，根据状态设置不同颜色

## 3. 数据处理建议

### 3.1 订单状态映射

后端返回的订单状态为英文，前端需要映射为中文：

| 后端状态 | 前端显示 | 标签类型 |
|---------|---------|---------|
| pending | 待付款 | warning |
| unshipped | 已付款 | primary |
| shipped | 已发货 | info |
| completed | 已完成 | success |
| cancelled | 已取消 | danger |
| refunding | 退款中 | warning |
| refunded | 已退款 | danger |

### 3.2 图表配置

**销售趋势图**:
- 折线图，显示销售额趋势
- 支持平滑曲线
- 添加区域填充效果
- 鼠标悬停显示具体日期和销售额

**订单状态分布图**:
- 饼图，显示各状态占比
- 添加图例
- 鼠标悬停显示具体状态、数量和占比

## 4. 性能优化建议

1. **数据缓存**: 对于仪表盘统计数据，可以在前端进行缓存，减少重复请求
2. **懒加载**: 图表组件可以使用懒加载，提高页面加载速度
3. **防抖处理**: 时间范围切换时使用防抖，避免频繁请求
4. **错误处理**: 对API请求失败的情况进行处理，显示友好的错误提示

## 5. 测试建议

1. 测试不同时间范围的销售趋势数据
2. 测试订单状态分布数据的准确性
3. 测试仪表盘统计数据的实时性
4. 测试页面响应式布局在不同屏幕尺寸下的表现

## 6. 技术栈

- 前端框架: Vue 3
- UI组件库: Element Plus
- 图表库: ECharts
- HTTP请求: axios

## 7. 接口调用示例

```javascript
// 获取仪表盘统计数据
async function getDashboardData() {
  try {
    const response = await axios.get('/api/admin/dashboard/statistics');
    if (response.data.code === 200) {
      return response.data.data;
    }
  } catch (error) {
    console.error('获取仪表盘数据失败:', error);
  }
}

// 获取销售趋势数据
async function getSalesTrend(timeRange = '7d') {
  try {
    const response = await axios.get('/api/admin/statistics/sales-trend', {
      params: { timeRange }
    });
    if (response.data.code === 200) {
      return response.data.data;
    }
  } catch (error) {
    console.error('获取销售趋势数据失败:', error);
  }
}

// 获取订单状态分布数据
async function getOrderStatusDistribution(timeRange = '7d') {
  try {
    const response = await axios.get('/api/admin/statistics/order-status', {
      params: { timeRange }
    });
    if (response.data.code === 200) {
      return response.data.data;
    }
  } catch (error) {
    console.error('获取订单状态分布数据失败:', error);
  }
}
```
